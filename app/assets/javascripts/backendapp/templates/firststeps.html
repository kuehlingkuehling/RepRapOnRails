Getting Started

This is the place where your RepRap Industrial 3D Printer gets its printjobs.

1. The Slicing Software

To prepare your digital 3D models for printing you need to install the Slic3r slicing software on your computer.
Follow these steps:

  • Go to http://slic3r.org/download and download the latest installation package for Slic3r for your operating system.
  • Follow the installation instructions available at
    http://manual.slic3r.org/GettingSlic3r.html#installing
  • Save the following preconfigured Slic3r profile for the RepRap Industrial to your computer:
    reprapindustrial_slic3r_profile.ini
  • In Slic3r go to File → Load Config and select your previously saved .ini file

2. Prepare your 3D model

Start Slic3r to process a 3D model into GCODE data. Select the preconfigured profiles for filament, print and printer before generating a GCODE file.
Refer to the Slic3r manual for help on finetuning any parameters for the printing process.

3. Add your printjob to the queue

Go to the Queue tab to upload your printjob to the 3D printer.

4. Start printing

The printjob is now available for printing on your RepRap Industrial and can be selected for printing via the touchscreen controller directly on the 3D printer.
